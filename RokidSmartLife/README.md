# 智慧生活 - Rokid AI 眼镜本地生活应用

> 灵感来自"高德地图携手雷鸟创新首发雷鸟智慧生活"，专为 Rokid AI 眼镜打造

## 功能特性

- **12大分类探索** — 美食、咖啡、购物、酒店、娱乐、景点、交通、医院、银行、超市、药店、加油站
- **扫街榜风格** — 附近 POI 按距离排序，显示评分、人均消费、距离
- **POI 详情** — 地址、电话、营业时间、评分等完整信息
- **步行/驾车导航** — 高德路线规划，逐步转弯指引
- **关键词搜索** — 搜索任意地点名称
- **D-pad 全适配** — 触控板操作，无需触摸屏

## 安装方式

```bash
adb install 智慧生活_v1.0.0_rokid.apk
```

### 方式二：分享 APK 给他人
直接将 `智慧生活_v1.0.0_rokid.apk` 文件发送给朋友，对方通过数据线 + ADB 安装即可。

## 首次使用设置

### 1. 连接 WiFi
眼镜需要连接 WiFi 才能获取定位和数据。通过 Rokid 手机 App 或 ADB 连接 WiFi。

### 2. 获取高德地图 API Key（必须）
1. 打开浏览器访问 https://lbs.amap.com
2. 注册/登录高德开放平台开发者账号
3. 进入控制台 → 应用管理 → 创建新应用
4. 添加 Key，服务平台选择 **「Web服务」**
5. 复制生成的 Key

### 3. 在应用中配置 Key
1. 打开「智慧生活」应用
2. 点击右上角 ⚙️ 进入设置
3. 粘贴/输入 API Key
4. 可选：调整搜索半径（500m~5km）
5. 点击保存

## 操作说明

| 操作 | 动作 |
|------|------|
| 选择分类/确认 | 触控板单击 |
| 上下移动 | 触控板上/下滑 |
| 左右移动 | 触控板左/右滑 |
| 返回上一页 | 触控板向左滑两次 |

## 界面说明

- **首页** → 12 个分类图标网格
- **分类列表** → 附近 POI 排行，显示排名、评分、价格、距离
- **POI 详情** → 完整信息 + 步行/驾车导航按钮
- **导航** → 逐步指引（显示方向箭头 + 文字描述）
- **搜索** → 输入关键词搜索地点
- **设置** → API Key 配置 + 搜索半径调整

## 技术信息

- Android 12 (API 32) 兼容
- 480×640 竖屏优化
- D-pad 焦点导航
- 暗色主题（AR 眼镜友好）
- 高德地图 Web Service REST API
- 轻量 APK (~13MB)

## 编译方式

```bash
cd RokidSmartLife
export JAVA_HOME=/opt/homebrew/opt/openjdk@17
export ANDROID_SDK_ROOT=/opt/homebrew/share/android-commandlinetools
./gradlew assembleDebug
```

APK 输出路径：`app/build/outputs/apk/debug/app-debug.apk`

## 注意事项

- 眼镜没有 GPS 芯片，定位依赖 WiFi 网络定位（精度约 100-500 米）
- 需要 WiFi 联网才能使用所有功能
- API Key 类型必须选择「Web服务」而非「Android平台」
- 首次启动需允许定位权限
