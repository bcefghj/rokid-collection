## 写法要点

1. **量化成果** — "识别准确率提升 4 倍"、"WER 1.74%"，让面试官有具体印象
2. **突出难点** — HAL 逆向分析、端侧 ASR 选型，说明你不是简单调 API
3. **技术深度** — 不要只说 "做了个 App"，要展示你解决了什么问题
4. **独立开发** — 强调独立完成全流程，面试时更有话语权
5. **GitHub 链接** — 简历中附上 GitHub 仓库链接，面试官可以看代码
