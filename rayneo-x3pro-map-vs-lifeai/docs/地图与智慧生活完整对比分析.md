## 七、代码层对比（C# DLL）

两个应用的 `ScriptingAssemblies.json` 中包含 **131个完全相同的DLL**，核心模块包括：

### 业务代码
- `Assembly-CSharp.dll` — 主业务逻辑
- `Assembly-CSharp-firstpass.dll` — 优先加载的业务代码

### 雷鸟SDK
- `RayNeo.dll` — 雷鸟SDK核心
- `Unity.XR.RayNeo.OpenXR.dll` — 雷鸟OpenXR插件
- `RayNeo.UnityAIOSAdapter.dll` — AIOS语音OS适配器
- `FFALFramework.dll` — ffalcon框架
- `UnityXRSDKCore.dll` — XR SDK核心

### 第三方库
- `DOTween.dll` / `DOTweenPro.dll` — 动画补间
- `Newtonsoft.Json.dll` — JSON序列化
- `LitJson.dll` — 轻量JSON
- `SoftMask.dll` — UI软遮罩
- `I18N.CJK.dll` — 中日韩字符集

### Unity框架
- 66个 `UnityEngine.*.dll` 模块（标准引擎组件）
- `Unity.XR.OpenXR.dll` — OpenXR插件
- `Unity.Localization.dll` — 本地化
- `Unity.InputSystem.dll` — 输入系统
- `Unity.TextMeshPro.dll` — 文字渲染

**结论：代码完全一致，差异仅在 Native 层和资源配置。**

### 8.2 智慧生活（新版）独有资源

```
assets/aa/Android/rayneolife_assets_all_*.bundle  — 雷鸟生活品牌资源
assets/eyrie/cloudres/*.xml (×499)                — 高德POI离线缓存（扫街榜数据）
```

### 8.3 解读

地图版本包含完整的 HERE 地图渲染方案（`here-custom-style.json` 达 20KB，定义了数百种地图元素的颜色方案：道路分级、建筑、水域、公园、军事区域等），这是一套专业级的暗色夜间地图样式，专为 AR 眼镜的半透明显示优化。

智慧生活版本砍掉了所有 HERE 内容，但新增了 499 个高德 POI 离线数据文件（`eyrie/cloudres/`），这些就是"扫街榜"功能的本地缓存——预加载附近高口碑商户信息，实现无需等待网络的即时显示。

## 九、演进关系总结

```
时间线：
────────────────────────────────────────────────────────
2026.01.29  地图 V2026.01.29.14 出厂预装
            ├── 包名: com.ffalcon.navigation（内部品牌）
            ├── 支持: 高德(中国) + HERE(海外) 双地图
            ├── 定位: 国际化版本
            └── 大小: 277MB

2026.03.11  AWE 2026 雷鸟×高德联合发布"雷鸟智慧生活"
            ├── 发布扫街榜、双模导航、LBS Agent
            └── 去掉海外地图，聚焦中国市场

2026.03.10  智慧生活 V2026.03.10.15 构建
2026.03.18  通过 Launcher OTA 推送安装到设备
            ├── 包名: com.rayneo.lifeai（消费者品牌）
            ├── 支持: 仅高德（新版v10.1.302 + Agent SDK）
            ├── 定位: 中国市场专版
            ├── 新增: 蓝牙定位辅助、扫街榜POI缓存
            └── 大小: 120MB（轻量化 57%）
────────────────────────────────────────────────────────
```

### 演进策略分析

1. **品牌统一**：从内部代号 `ffalcon` 转向消费者品牌 `rayneo`
2. **市场聚焦**：从国际化（HERE+高德）转向中国专版（仅高德）
3. **SDK升级**：高德SDK跨大版本升级（v9→v10），新增Agent能力
4. **功能增强**：新增蓝牙定位辅助、扫街榜离线数据、镜腿按键支持
5. **体积优化**：去掉72MB HERE SDK，整体减小57%
6. **部署方式**：从系统预装改为 Launcher OTA 推送，更新更灵活
