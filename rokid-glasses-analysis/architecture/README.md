## 通信架构

```
┌──────────────────┐              ┌──────────────────┐
│    Rokid 眼镜     │              │     手机 App      │
│                  │              │                  │
│ AssistServer     │              │ AI 服务          │
│   ↕ AIDL        │   蓝牙/WiFi   │ 翻译服务         │
│ CXRService ─────│─────────────→│ 导航服务         │
│   ↕ Caps协议    │              │ 支付代理         │
│ SysConfig       │←─────────────│ 通知转发         │
│   ↕ UEvent      │   3种连接方式: │ 直播服务         │
│ 硬件传感器       │   1.BT RFCOMM│ 音乐控制         │
│                  │   2.BLE GATT │                  │
│                  │   3.MFI iAP2 │                  │
└──────────────────┘              └──────────────────┘
```

## 场景管理架构

```
SceneCoreManager (场景核心管理器)
  │
  ├── Scene (功能场景) × 18
  │   ├── translate (翻译)
  │   ├── navigation (导航)
  │   ├── ai_chat (AI聊天)
  │   ├── video_record (录像)
  │   ├── audio_record (录音)
  │   ├── live_broadcast (直播)
  │   ├── phone_call (电话)
  │   ├── mobile_music (音乐)
  │   ├── music_word (歌词)
  │   ├── word_tips (提词器)
  │   ├── payment (支付)
  │   ├── food_order (点餐)
  │   ├── city_guide (城市导览)
  │   ├── custom_view (自定义)
  │   └── ...
  │
  ├── Page (页面) × 5
  │   ├── launcher_main (主页)
  │   ├── settings (设置)
  │   ├── brightness (亮度)

  │
  └── Cut (快捷操作) × 2
      ├── take_picture (拍照)
```
