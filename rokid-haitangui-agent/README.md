# 🐢 海龟汤推理师 · Rokid 灵珠智能体

> 基于 Rokid AR 眼镜的语音推理游戏智能体，用户通过是非问句还原离奇故事真相。

## 项目简介

"海龟汤"是一种经典的情景推理游戏：主持人讲述一个离奇的故事片段（汤面），玩家通过不断提问（只能回答是/否/无关）来推理出完整真相（汤底）。

本智能体将这个游戏搬到 Rokid AR 眼镜上，实现**纯语音驱动**的沉浸式推理体验。

## 功能特性

- 🎯 **难度分级**：简单 / 中等 / 困难三档，适合不同用户
- 🗣️ **语音适配**：所有回复控制在 50 字以内，专为语音播报优化
- 📚 **100 道题库**：涵盖经典、温馨、搞笑、恐怖多种风格
- 🔒 **严格规则**：AI 只回答"是的/不是/无关/问题无效"，绝不提前泄露答案
- 🔄 **灵活控制**：支持换题、放弃、追加难度等语音指令

## 快速开始

### 使用方式（灵珠平台）

1. 登录 [灵珠开发者平台](https://agent-develop.rokid.com)
2. 创建知识库，上传 `海龟汤题库_100题.md`（按 `---` 分割）
3. 新建对话型智能体，名称填"海龟汤推理师"
4. 将 `Prompt_系统提示词.md` 中的基础版 Prompt 粘贴到系统提示词
5. 关联知识库，发布，在 Rokid AI App 中启用

### 语音交互示例

```
用户：来一道海龟汤
AI：好的，你想要什么难度？简单、中等还是困难？
用户：中等
AI：[朗读汤面故事]
用户：是不是发生在医院？
AI：不是
用户：是不是有人死亡？
AI：是的
用户：我猜到了，是因为...
AI：恭喜你猜对了！真相是[朗读完整汤底]
```

## Prompt 设计亮点

| 特性 | 说明 |
|------|------|
| 角色锁定 | AI 始终扮演"出题人"，严格按汤底判断，不会被套话 |
| 回复限制 | 50 字硬性上限，语音播报不超过 15 秒 |
| 多版本 | 基础版 / 带积分进阶版 / 多人对战版，按需选用 |
| 防剧透 | 用户猜对 80% 核心要素才算过关 |

## 题库说明

- 总计 **100 道**题目
- 每道题包含：汤面（谜面）+ 汤底（真相）
- 分类：简单 30 题 / 中等 40 题 / 困难 30 题
- 格式：Markdown，每题用 `---` 分隔，便于灵珠平台知识库切割
