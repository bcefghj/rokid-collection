# 海龟汤推理师 · 开发操作手册

> 平台：Rokid 灵珠平台（agent-develop.rokid.com）  
> 目标：在 Rokid Glasses 上运行一个纯语音海龟汤推理智能体  
> 难度：零代码，纯可视化操作

## 第一步：注册开发者账号

### 1.1 注册 Rokid 开放平台

1. 打开浏览器，访问：**https://developer.rokid.com**
2. 点击右上角「注册」
3. 填写手机号，完成验证码注册
4. 登录后进入「实名认证」页面，上传身份证完成认证
   - ⚠️ 实名认证是必须的，否则无法使用 API 权限

### 1.2 申请灵珠开发者权限

1. 打开：**https://agent-develop.rokid.com**
2. 使用与 **Rokid AI 手机 App 相同的账号** 登录
   - ⚠️ 必须是同一个账号，否则手机 App 里看不到你创建的智能体
3. 首次进入会提示申请开发者资格，按提示填写信息提交即可
4. 审核一般当天通过

## 第二步：创建知识库（题库）

> 知识库让智能体能「记住」100道题，不用全部塞进 Prompt

### 2.1 进入知识库页面

1. 登录灵珠平台后，左侧菜单点击「**知识库**」
2. 点击右上角「**新建知识库**」

### 2.2 填写基础信息

| 字段 | 填写内容 |
|------|----------|
| 知识库名称 | 海龟汤题库 |
| 描述 | 包含100道海龟汤题目，含汤面、汤底、难度标签 |

点击「**确认创建**」

### 2.3 上传题库文件

1. 进入刚创建的知识库，点击「**上传文档**」
2. 选择本地文件：`海龟汤题库_100题.md`
3. 等待解析完成（通常1-3分钟）
4. 解析完成后，可以在右侧预览看到分段内容

### 2.4 配置切分规则（重要）

上传完成后，点击文件右侧「**设置**」，配置分段规则：

| 设置项 | 推荐值 |
|--------|--------|
| 分段方式 | 按自定义分隔符 |
| 分隔符 | `---`（三条横线，即每道题之间的分隔） |
| 最大分段长度 | 500 字符 |

点击「**保存并重新分段**」，等待处理完成。

## 第三步：创建智能体

### 3.1 新建智能体

1. 左侧菜单点击「**智能体**」
2. 点击「**创建智能体**」
3. 填写基础信息：

| 字段 | 填写内容 |
|------|----------|
| 智能体名称 | 海龟汤推理师 |
| 描述 | 一款语音推理游戏，通过是非问答引导用户推理出隐藏真相 |
| 头像 | 上传本文件夹中的图标（可选） |

4. 点击「**确认**」

### 3.2 配置模型

进入智能体编辑页，找到「**模型**」设置：

| 设置项 | 推荐值 |
|--------|--------|
| 模型选择 | 推荐 `Qwen-Max` 或平台默认模型 |
| 温度（Temperature） | 0.7（适当创意，不要太随机） |
| 最大回复长度 | 300 tokens（语音播报，回复要简洁） |

### 3.3 填写系统提示词（Prompt）

这是最核心的步骤，找到「**系统提示词**」输入框，复制以下内容粘贴进去：

```
你是一位神秘的海龟汤出题人，精通逻辑推理游戏。

【角色设定】
你掌握一个完整的故事（汤面+汤底），用户只知道汤面，需要通过提问还原真相。
你只能回答以下四种之一：
- "是的"
- "不是"
- "这和案件无关"
- "问题无效，请换一种方式提问"

【游戏规则】
1. 用户说"开始游戏""来一道海龟汤""我要玩"等，先问难度（简单/中等/困难）
2. 根据难度从知识库中选择一道题，朗读汤面给用户
3. 用户用是非句提问，你严格按照汤底判断，只给上述4种回答之一
4. 绝对不能主动透露任何汤底信息
5. 当用户说"我猜到了"或"答案是..."，判断是否猜中核心要素（80%以上即算对）
   - 猜对：恭喜用户，完整朗读汤底真相
   - 猜错：回复"还差一点点，继续加油！可以换个角度想想～"
6. 用户说"放弃""揭晓答案""我不猜了"：直接朗读完整汤底
7. 用户说"换一道""再来一题"：询问是否更换难度，然后出新题

【语气风格】
- 神秘感十足，语气简短有力
- 回复必须简洁，适配语音播报（每次回复不超过50字）
- 可以适时鼓励："你已经很接近了！""这个方向很对！"
- 开场时使用以下话术：

"欢迎来到海龟汤推理室。我手中有几个离奇的故事，表面简单，真相却出人意料。请告诉我你想要什么难度：简单、中等、还是困难？"

【注意事项】
- 回复语言简洁，避免长段文字
- 每次只回答一个问题
- 不要在回复中加任何表情符号
```

### 3.4 关联知识库

在智能体编辑页，找到「**知识库**」区域：
1. 点击「**添加知识库**」
2. 选择刚才创建的「海龟汤题库」
3. 配置检索设置：

| 设置项 | 推荐值 |
|--------|--------|
| 检索策略 | 混合检索 |
| Top K（返回条数） | 3 |
| 相似度阈值 | 0.5 |

4. 点击「**确认**」

### 3.5 配置开场白

找到「**开场白**」设置，填写：

```
欢迎来到海龟汤推理室。我手中有几个离奇的故事，表面简单，真相却出人意料。你准备好了吗？请告诉我你想要什么难度：简单、中等、还是困难？
```

### 3.6 配置快捷指令（可选）

找到「**快捷指令**」区域，添加以下常用指令：

| 指令名 | 指令内容 |
|--------|----------|
| 开始游戏 | 我要玩海龟汤，给我来一道 |
| 简单题 | 给我一道简单的海龟汤 |
| 困难题 | 给我一道最难的海龟汤 |
| 揭晓答案 | 我放弃了，揭晓答案 |
| 换一道 | 这道不玩了，换一道 |

## 第四步：调试测试

### 4.1 平台内调试

在智能体编辑页右侧，有一个「**调试对话**」面板：

1. 在输入框输入：`开始游戏`
2. 检查智能体是否询问难度
3. 选择难度后，检查是否正确朗读汤面
4. 提问几个是非题，检查回答是否准确
5. 尝试猜测答案，检查判断逻辑

**重点检查清单：**
- [ ] 开场白是否自然
- [ ] 能否正确抽取对应难度的题目
- [ ] 是非回答是否简洁（不超过50字）
- [ ] AI 有没有"漏嘴"提前透露汤底关键信息
- [ ] 猜对猜错的判断逻辑是否合理
- [ ] 用户说"放弃"时能否正确揭示汤底

### 4.2 常见问题处理

**问题1：AI 回答太长，不适合语音播报**
- 解决：在 Prompt 末尾加强："每次回复必须控制在30字以内，这是硬性要求"

**问题2：AI 主动透露了汤底**
- 解决：在 Prompt 中加："在用户猜中之前，无论如何都不得透露汤底任何细节"

**问题3：AI 选题不准确，选的题难度不对**
- 解决：检查知识库分段是否正确，确保每道题包含难度标签

**问题4：AI 无法判断用户是否猜对**
- 解决：在 Prompt 中加："判断用户答案时，关注核心要素是否命中，不要求用词完全一致"

## 第五步：发布到 Rokid Glasses

### 5.1 保存智能体

点击右上角「**保存**」按钮，确认保存成功。

### 5.2 手机 App 端操作

1. 打开手机上的 **Rokid AI App**
2. 确保手机账号与灵珠平台账号一致
3. 进入：**设置 → 开发者 → 智能体调试**
4. 你会看到刚才创建的「海龟汤推理师」出现在列表中
5. 点击「**进入**」

> ⚠️ 注意：未提审的智能体只有自己的账号能看到，不要点「提审」按钮

### 5.3 连接眼镜测试

1. 打开 Rokid Glasses，确保与手机蓝牙连接正常
2. 在手机 App 的智能体调试页面，点击你的智能体
3. 戴上眼镜，说出唤醒词或点击开始
4. 说"开始游戏"，测试完整流程

### 5.4 配置快捷唤醒（推荐）

在 Rokid AI App 中：
1. 进入「**AI 快捷指令**」设置
2. 添加新指令：
   - 指令词：`海龟汤`（或任何你喜欢的词）
   - 对应智能体：海龟汤推理师
3. 保存后，直接对眼镜说"海龟汤"即可快速启动

## 第六步：后续优化方向

### 优化1：增加多人模式
在 Prompt 中添加：
```
用户说"多人模式"或"我们一起玩"时，切换为裁判模式：
你出题后，不同的人可以轮流提问，你统一回答"是的/不是/无关"。
```

### 优化2：增加历史记录
在灵珠平台开启「**对话历史**」功能，让智能体记住本次已经玩过的题，不重复出题。

### 优化3：增加积分系统
在 Prompt 中添加问题计数逻辑：
```
每次开始新题时，记录计数为0。
每次用户提问，计数+1。
用户猜对后，告知本题用了X个问题，并评价：
- 1-5题：天才推理师！
- 6-15题：优秀侦探！
- 16题以上：慢慢来，下次会更好！
```

### 优化4：扩充题库
按照现有格式，在 `海龟汤题库_100题.md` 中继续添加题目，然后重新上传到知识库即可。
