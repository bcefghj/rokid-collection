## 追问示例对话

```
用户：讲一下15题
AI：三数之和。给定数组，找出所有和为0的不重复三元组。
    核心思路：排序后固定第一个数，用左右双指针在剩余部分找两数之和，注意跳过重复元素，时间复杂度O(n²)。

用户：为什么要排序？
AI：排序有两个作用：一是双指针需要有序数组才能正确收缩；二是相邻相同元素放在一起，方便通过跳过相邻重复值来去重，避免结果中有重复三元组。

用户：给我看代码
AI：[给出完整Python代码]

用户：时间复杂度怎么算
AI：排序是O(n log n)，外层循环O(n)，内层双指针O(n)，总体O(n²)。空间复杂度O(1)，不需要额外数据结构。

用户：有没有更好的解法
AI：三数之和的最优解就是O(n²)，这已经是理论下界了。暴力枚举是O(n³)，哈希表方法虽然也能做，但去重逻辑复杂，实际不比双指针快。
```
